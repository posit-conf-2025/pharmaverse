Tables in R using {tfrmt}

ARD-first Tables with {tfrmt}

  • Metadata-driven table formatting

  • Easily create new and modify existing tables

  • Input: ARD with raw, numeric values ({cards}!)

  • Output: Formatted table via {gt}

The {tfrmt} object

  • Pre-define the non-data components of your table
  • Pre-define how the data will be handled once added

cards to tfrmt - How did we get here?

{tfrmt} includes several helper functions to transform native {cards} output to display-ready data. Functionality includes:

  • Coalesce/rename the group and variable names/levels
  • Unlist the list-columns
  • Prepare row group and label columns
  • Fill missing variable values in a hierarchy
  • Prepare big N rows for headers

To explore, let’s start with the result of our cards::ard_stack() call.

library(cards)
 
ard_demog <- cards::ard_stack(
  data = adsl,
  .by = TRT01A,
  cards::ard_continuous( variables = AGE,
                         statistic = ~ cards::continuous_summary_fns(c("median", "p25", "p75"))),
  cards::ard_categorical(variables = c(AGEGR1, SEX), statistic = ~ c("n","p")),
  .overall = TRUE,
  .total_n = TRUE
)

  tfrmt::prep_big_n(vars = "TRT01A")
